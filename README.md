# ALDORITHM_LAB3

Було реалізовано виведення окличних речень у прямому та зворотному порядку.
Речення зчитуються з файлу "file.txt". Результат записується у файл "output.txt".
Дане завдання було реалізовано у двох варіантах: з використанням
ітераційного алгоритму та з використанням рекурсії.
Збільшуючи кількість елементів (глибину рекурсії), було отримано переповнення стеку.
Окрім цього, визначено час виконання ітераційного алгоритму та з використанням рекурсії для кількості
елементів 100, 5000 та для n, близького до Stack Overflow

##Таблиця результатів:

| К-сть елементів \ алгоритм    | Ітерація        |  Рекурсія     |
|:----------------------------- |:---------------:| -------------:|
| 100                           | 0               |     0,037     |
| 5000                          | 0               |        0,121  |
| (n > 39_000_000 -> ?) / (5992)| 0,04            |       0,064   |
